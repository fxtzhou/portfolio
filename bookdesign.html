Book Design

Using the text and images supplied in Paul Rand's Design and the Play Instinct, we were asked to develop a strong visual concept and utilize a grid system to create a book with constraints.

Thumbnail sketches and spread sketches really guided me on this project since spreads had to be consistent in design. The use of grid was especially important so that the layout would be cohesive. In the beginning, I had a lot of ideas and really wanted to be playful by using imagery from the images given and have that inspire the layout of the text that followed since the emphasis of the text is on the importance of designs.
