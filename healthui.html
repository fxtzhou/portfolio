Healthcare User Interface

The purpose of this assignment was to create a homepage interface, or make an existing one better for one of several categories. I was assigned healthcare, and created a interface based on existing trends that I saw in healthcare companies using the Sketch App. We were required to follow the UX process such as comparative research, creating personas, and reiterating through feedback.

Process and Drafts

Created a persona that would look for an alternative-type of health care provider.


The first wireframe had placeholder lorem ipsum text and a fake title as it was being developed.

In a future iteration, a login was placed along with a more conventional horizontal navigation bar. More colors were used to stray away from the 'medical blue' vibe the first iteration gave off. Yellow was used instead to convey a comforting, hopeful feeling.
